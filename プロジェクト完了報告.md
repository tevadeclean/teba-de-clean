## 今後の運用方法

### サイトの更新方法

#### 方法1: GitHubで直接編集（簡単）
1. https://github.com/tevadeclean/teba-de-clean にアクセス
2. 編集したいファイルを開く
3. 右上の鉛筆アイコンをクリックして編集
4. 「Commit changes」をクリック
5. 数分後、自動的にサイトに反映されます

#### 方法2: ローカルで編集（高度）
1. リポジトリをクローン: `git clone https://github.com/tevadeclean/teba-de-clean.git`
2. ファイルを編集
3. コミット＆プッシュ: `git add . && git commit -m "更新内容" && git push`
4. 自動的にVercelにデプロイされます
